About Me Biography

Hello, I'm Andrew aka "whiteex" and I'm a 19 years old beginner programmer from Romania.


I work on a lot of different projects, such as the Discord bot "Eutera" and my Unreal Engine plugins "conSole" and "WiMan".

Projects Decay

Decay is an atmospheric in-development game, built on Unreal Engine 5, with a demo version released in late 2019 on Unreal Engine 4.


Currently there is no known release date for this project.

Projects Eutera

Eutera was built with the help of NodeJS, with it's first version being released somewhere around late 2020.


Eutera (4.5.2) has been released in March 14th 2022 and is the latest version to date.
